
# 🏥 MediCare Plus — Your Health, Our Priority

A fully responsive healthcare website for **MediCare Plus**, a multi-specialty medical clinic serving patients since 2005.

## 🌐 Live Demo
👉 https://demomedicareplus.netlify.app/

## 📋 Features
- 🏠 Hero section with appointment CTA
- 🩺 6 Medical Specialties — Cardiology, Neurology, Dental, Ophthalmology, Orthopedics & Pediatrics
- 👨‍⚕️ Doctor profiles with credentials
- ⭐ Patient testimonials carousel
- ❓ FAQ accordion section
- 📅 Appointment booking form
- 📱 Fully responsive design
- 💬 Telehealth / Video visit support info
